# Multimodal Perception Module (Lab 2)

This repository contains the `master_fusion.py` script, which serves as the core multimodal perception pipeline for our empathetic Conversational Agent (CA). It concurrently captures and processes lexical, acoustic, and visual data from the user to construct a comprehensive emotional profile, which is then injected into a local LLM Dialog Manager.

## Features
* **Multimodal Capture:** Concurrently records audio (16kHz) and video (1 fps) during a user's conversational turn using threading.
* **Lexical Modality:** Uses `Faster-Whisper` (or `openai/whisper-small.en`) for transcription and a RoBERTa-based model (`SamLowe/roberta-base-go_emotions`) to extract 28 nuanced text emotions.
* **Acoustic Modality:** Uses the `wav2small` Audeering model to extract continuous prosodic features (Arousal, Dominance, Valence) from the audio waveform.
* **Visual Modality:** Uses `DeepFace` to detect facial landmarks and output probability distributions across Ekman’s basic emotions, averaged across all valid frames captured during the turn.
* **Dialog Manager Integration:** Injects the fused emotional context and transcription directly into a local instance of Llama 3 (via Ollama) to generate empathetic responses.
* **Latency Benchmarking:** Automatically tracks and outputs the inference time for each sub-module to evaluate system scalability.

## Prerequisites

1. **Python Packages**: Ensure you have the required libraries installed
    ```
    pip install -r requirements.txt
    ```
2. **Ollama (Llama 3)**:
You must have Ollama installed and running locally with the Llama 3 model downloaded

    ```
    ollama run llama3
    ```

## How to Run
Open your terminal and execute the script:

```
python master_fusion.py
```

The script will initialize the webcam and load the ML models into memory (this may take 10-15 seconds).

When prompted, press ``[ENTER]`` to begin your conversational turn. Speak naturally into your microphone while looking at the camera.

Press ``[Ctrl+C]`` when you are finished talking.

### Expected Output
Once recording stops, the script will sequentially run the classifiers. It will output:

- The AI agent's empathetic response generated by Llama 3.
- A breakdown of the detected emotional state across all three modalities (Text, Audio, Video).
- A latency benchmark report detailing the processing time for each individual model and the total pipeline latency.

Note: For debugging purposes, the script also saves the captured video frames in a debug_frames/ directory.
